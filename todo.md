- [x] Добавить возможность загружать и прикреплять фотографии к грядкам в огороде (сохранение в Supabase Storage `garden_photos`).
- [x] Добавить отдельный справочник препаратов с описанием, аналогами, химическим составом и дозировками с переходом из карточек ухода у растений.

- [x] все операции удаления и другие важные действия обязательно через окно диалога нашего для подтверждения
- [x] Чтобы при прокрутке длинного списка карточек ухода или секретов можно было быстро вернуться к самому началу, предлагаю добавить плавающую кнопку «Наверх» (Scroll to top) ⬆️, которая будет появляться в правом нижнем углу после прокрутки первых 500px.
- [x] Чтобы визуально отделить зафиксированную шапку растения от прокручиваемого списка внизу, предлагаю добавить динамическую тень (Glassmorphism shadow), которая будет плавно усиливаться по мере пролистывания контента вниз.
- [x] расширить справочник препаратов и удобрений
- [x] Интеграция Агрокалькулятора с Журналом ухода: Можно добавить в карточку результата калькулятора кнопку «Записать в журнал полива/подкормки». При нажатии на неё приложение автоматически откроет форму добавления записи (AddTreatmentView), где в качестве выполненного действия уже будут указаны рассчитанные данные (например: «Подкормка: Монофосфат калия, 50г на 25л воды»). Это полностью автоматизирует ведение дневника и сэкономит массу времени!
